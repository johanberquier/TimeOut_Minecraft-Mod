# TimeOut_Minecraft-Mod
TimeLock régule l'accès à votre serveur Minecraft selon l'heure réelle. Les connexions sont autorisées uniquement de 12h00 à 02h00 du matin. En dehors, l'accès est rejeté avec un message personnalisé. Fini le farming nocturne intensif : tout le monde progresse au même rythme pour garantir une expérience communautaire 100 % équitable.
